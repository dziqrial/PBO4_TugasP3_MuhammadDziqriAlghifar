a. Scanner

Termasuk di paket java.util.

Langsung bisa membaca berbagai tipe data (nextInt(), nextDouble(), nextLine(), dll).

Lebih mudah digunakan tapi sedikit lebih lambat karena melakukan parsing tipe data otomatis.

Contoh deklarasi:

Scanner sc = new Scanner(System.in);


b. BufferedReader

Termasuk di paket java.io.

Hanya bisa membaca input dalam bentuk String, sehingga harus dikonversi manual ke tipe data lain menggunakan Integer.parseInt() atau Double.parseDouble().

Lebih cepat karena menggunakan mekanisme buffer untuk membaca data.

Cocok untuk input besar atau proses I/O yang berulang.

Contoh deklarasi:

BufferedReader br = new BufferedReader(new InputStreamReader(System.in));


c. Perbandingan Umum

Scanner:
Paket	java.util
Jenis Input: Bisa langsung tipe data (int, double, dll)	
Kecepatan: Sedikit lebih lambat
Kemudahan: Simpel untuk pemula
Penggunaan Umum:Input sederhana

BufferedReader:

Paket	java.io
Jenis Input: Harus String, lalu diubah manual
Kecepatan: Lebih cepat
Kemudahan: Perlu parsing tambahan
Penggunaan Umum:Input besar/file
